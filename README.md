# SCREENL — WordPress Theme

Одностраничный лендинг аренды LED-пилонов (1:1 с вёрсткой `new/`).

## Установка

1. Скопируйте папку `screenl` в `wp-content/themes/screenl/`.
2. В админке: **Внешний вид → Темы** → активируйте **SCREENL**.
3. **Настройки → Чтение** → «Статическая страница» → создайте страницу «Главная» и выберите её.
4. Откройте сайт — отобразится `front-page.php` с полным лендингом.

## Заявки (email)

Формы отправляют данные через `admin-ajax.php` и `wp_mail()` на адрес администратора WordPress.

Чтобы указать другой email, добавьте в `wp-config.php`:

```php
define('SCREENL_LEAD_EMAIL', 'rent@screenl.ru');
```

На продакшене настройте SMTP (плагин **WP Mail SMTP** или аналог), иначе письма могут не доходить.

## Структура

- `front-page.php` — главная
- `template-parts/sections/` — секции лендинга
- `template-parts/overlays/` — меню и модалка
- `assets/`, `js/`, `styles.css`, `fonts.css` — статика как в `new/`
- `inc/` — подключение ресурсов, обработчик форм, очистка стилей WP

## Проверка после установки

- Hero: анимация панелей
- Scenarios: pin-stack при скролле
- Configs: слайдер 1–6 пилонов
- Menu / modal / формы
- Масштаб на ширине &lt; 1920px (`zoom`)

Сравните с `new/index.html` на тех же разрешениях (1920, 1440, 1280).
